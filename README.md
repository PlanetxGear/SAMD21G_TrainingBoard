# Microchip SAMD21G Training Board for 「SAMファミリ活用ガイドブック」.
---
## Description
後閑哲也氏の著書[ＳＡＭファミリ活用ガイドブック][1](ISBN 978-4-297-11291-2)に出てくるトレーニングボードをKiCADで書き起こした回路図とガーバーデータです。
これは[KiCAD][2]によって描かれています。　この資料が皆さんの学習の一助となれば幸いです。
